Projects Here you will find some of the personal and schools projects that I created with each project containing its own case study

Holberton School - Simple Shell

This project consists of creating a simple shell program written in C that mimics the basic behavior of a UNIX shell. The shell provides a command-line interface where users can enter standard system commands. The goal of this project is to deepen understanding of low-level system programming concepts in a UNIX-like environment.

Simple Shell

Holberton School - HbnB

The Holberton School HBNB Project is a web development project where students create a simple Airbnb clone using various technologies, such as Python, Flask, and SQLAlchemy. It is part of the Full Stack Software Engineering curriculum at Holberton School.

HbnB

App IAKOA - iOS

IAKOA is an iOS application designed to help you discover events happening around you. With an intuitive interface, geolocation features, and the ability to create your own events, it makes it easy to stay connected to what's going on nearby and contribute to your local community
